Đề xuất lộ trình (dự kiến 18 bài, 4 giai đoạn)

Giai đoạn 1 — Nền tảng (Bài 1-5): Cấu trúc app, widget cơ bản, layout, Signal/Slot, xử lý sự kiện

Giai đoạn 2 — Widget nâng cao & UI chuyên nghiệp (Bài 6-10): Model/View, QTableView + dữ liệu, Dialog, Menu/Toolbar/StatusBar, Style (QSS - CSS cho Qt)

Giai đoạn 3 — Kiến trúc & hiệu năng (Bài 11-15): QThread/đa luồng, kết nối cơ sở dữ liệu (SQLite/PostgreSQL — nối với kiến thức DB bạn đang học), Qt Designer + .ui file, đóng gói ứng dụng (PyInstaller)

Giai đoạn 4 — Dự án thực tế (Bài 16-18): Xây dựng 1 ứng dụng desktop hoàn chỉnh (ví dụ: app quản lý đơn hàng kết nối PostgreSQL, có đa luồng, có model/view)
